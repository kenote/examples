## Examples

- `webpack-ts` - Use `Webpack` to compile `Typescript` style `Javasctipt` modules.
- `module-ts` - Use `tsc` to compile `Typescript` style `Javasctipt` modules.

## Usages

Installing all modules
```bash
$ yarn bootstrap
```
